دوام (דוואם) — جاهز ✅

في هذا الإصدار:
- عبري افتراضي + زر لغة (عربي)
- صفحة مدير لإضافة موظفين (اسم كامل + رقم + كلمة مرور)
- تسجيل دخول/خروج + إشعار إيميل لصاحب المتجر
- بعد تسجيل الخروج يظهر للموظف:
  1) ساعات المناوبة
  2) مجموع ساعات آخر 30 يوم
  3) الأيام اللي اشتغلها (آخر 30 يوم) مع ساعات كل يوم

========================
التشغيل (Windows)
========================
1) فك الضغط
2) افتح PowerShell داخل: dawam/server
3) شغّل:
   npm install
   npm start
4) افتح:
   الموظف: http://localhost:3000
   المدير:  http://localhost:3000/admin.html

========================
مهم جداً (App Password)
========================
Google App Password أحيانًا يظهر بمسافات (مثل: "iihr dmlx akyw ecnn")
داخل .env لازم يكون بدون مسافات، لذلك تم حفظه هكذا:
SMTP_PASS=iihrdmlxakywecnn

========================
لو عندك قاعدة بيانات قديمة
========================
احذف: server/dawam.sqlite
وبعدين شغّل السيرفر من جديد.
